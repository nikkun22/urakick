## はじめに（このサーバーは何のためのもの？）

裏キック団のメンバー10名がKickで同時に配信したとき、よく起こる問題があります。

- 📵 **電波が弱くなると配信が止まる**（特に外配信中）
- 🐢 **遅延が大きくて視聴者のコメントが追いつかない**
- 💥 **何度も配信が落ちて視聴者がストレスを感じる**

これを解決するために「**中継サーバー**」を間に入れます。

```
（今までの配信）
  メンバー → Kick           ← 電波が悪いと即落ちる

（中継サーバーありの配信）
  メンバー → 中継サーバー → Kick  ← サーバーが受け止めて安定化
```

中継サーバーは「電波の悪さを吸収するクッション」のような役割です。
途切れそうになっても、サーバーが踏ん張って配信を継続させてくれます。

---

## メンバーの配信ツールと対応状況

10名のメンバーは以下のいずれかで配信されると想定しております。

| 配信ツール | 中継サーバー対応 | 説明 |
|---|---|---|
| **Moblin**（iPhone用アプリ） | ⭕ 対応 | 4G・5G・WiFiを束ねて使え、外配信に最強 |
| **OBS**（パソコン用ソフト） | ⭕ 対応 | 自宅・スタジオ配信の定番ソフト |
| **KICK公式アプリ**（スマホ用） | ❌ 非対応 | Kickに直接配信される仕様のため、中継不可 |

### つまりどういうことか

- **Moblin / OBS でご配信の方** → 中継サーバーで安定化されます
- **KICK公式アプリでご配信の方** → 今まで通り直接配信となります

KICK公式アプリのみのメンバーがいる場合、その方は中継サーバーを使わずそのまま配信していただきます。中継サーバーが必要なのは Moblin / OBS を使う方のみです。

## ご提案プラン（2案）

サーバーの構築方法には大きく2つあります。

### 【A案】ハイブリッド構成（おすすめ）

「**インターネット上のレンタルサーバー**」と「**自宅の小型サーバー**」を組み合わせる方式です。

```
[メンバー] → [レンタルサーバー（窓口）] → [自宅の小型サーバー（処理）] → [Kick]
              ↑
         この窓口だけが世間に公開され、
         自宅サーバーは外から見えない（安全）
```

#### なぜ2つに分けるのか

- **レンタルサーバー**: 世間に公開する「受付窓口」役。住所を隠せる・攻撃から守れる
- **自宅小型サーバー**: 実際の処理を担当。月額コストがほぼゼロで動く
